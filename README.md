## ПЗ №6 Составление алгоритма решения задачи.

### Задача 1. Перевод времени

Ввод: количество часов (h)

Вычисление: минуты=h×60

Вывод: количество минут

Пример: при вводе h=2, вывод: 120 минут

### Задача 2. Сравнение чисел

Ввод: числа A и B

Алгоритм:

Если A>B, то вывод: «Число A больше»

Иначе вывод: «Число B больше»

Пример: при A=5, B=8, вывод: «Число B больше»

### Задача 3. Допуск к экзамену

Ввод: оценка (o)

Алгоритм:

Если o≥3, то вывод: «Допущен»

Иначе вывод: «Не допущен»

Пример: при оценке o=4, вывод: «Допущен»

### Задача 4. Скидка в магазине

Ввод: цена за единицу (p), количество (q)

Вычисления:

общая_сумма=p×q

Если общая_сумма>1000, то:

к_оплате=общая_сумма×0,9 (применяется скидка 10 %)

Иначе:

к_оплате=общая_сумма (скидка не применяется)

Вывод: «К оплате: к_оплате руб.»

Примеры:

При p=200, q=6:

общая_сумма=200×6=1200

Так как 1200>1000, применяем скидку: к_оплате=1200×0,9=1080

Вывод: «К оплате: 1080 руб.»

При p=100, q=5:

общая_сумма=100×5=500

Так как 500≤1000, скидка не применяется: к_оплате=500

Вывод: «К оплате: 500 руб.»

### Задача 5. Решение квадратного уравнения 

Ввод: a, b, c 

Проверка: Если a = 0: Вывод: "Уравнение не является квадратным" 

Иначе: Вычисление дискриминанта: D = b² - 4ac

Анализ: Если D < 0: Вывод: "Корней нет"

Если D = 0: x = -b / (2a) Вывод: "Один корень: " + x

Если D > 0: x1 = (-b + √D) / (2a) x2 = (-b - √D) / (2a) Вывод: "Корни: x1 = " + x1 + ", x2 = " + x2

Пример:

При a = 1, b = -4, c = 4: вывод: "Один корень: 2"

При a = 1, b = 1, c = 1: вывод: "Корней нет"

При a = 0, b = 2, c = 5: вывод: "Уравнение не является квадратным"
